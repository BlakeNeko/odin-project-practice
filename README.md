# odin-project-practice

A series of practice projects for the Odin Project.

## Description of repository structure

**odin-recipes**: code for [Project: Recipes](https://www.theodinproject.com/lessons/foundations-recipes)

**landing-page**: code for [Project: Landing Page](https://www.theodinproject.com/lessons/foundations-landing-page)

**rock-paper-scissors**: code for [Project: Rock Paper Scissors](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors)

**rock-paper-scissors-ui**: same project as [Project: Rock Paper Scissors](https://www.theodinproject.com/lessons/foundations-rock-paper-scissors), but with a UI

**etch-a-sketch**: code for [Project: Etch-a-Sketch](https://www.theodinproject.com/lessons/foundations-etch-a-sketch)

**calculator**: code for [Project: Calculator](https://www.theodinproject.com/lessons/foundations-calculator)
